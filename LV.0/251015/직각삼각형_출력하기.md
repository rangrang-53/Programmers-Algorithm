# [Level 0] 직각삼각형 출력하기

## 📝 문제 설명 (요약)  
자연수 `n`이 주어질 때,  
높이가 `n`인 **직각삼각형 모양의 별(`*`)**을 출력하라.

- 제한사항  
  - 1 ≤ `n` ≤ 10  

- 반환값: 없음  
  - 표준 출력(`System.out.println`)을 사용하여 별을 출력해야 함.

## 💡 예시
입력:
```
3
```

출력:
```
*
**
***
```
