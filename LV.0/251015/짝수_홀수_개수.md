# [Level 0] 짝수 홀수 개수

## 📝 문제 설명 (요약)  
정수 리스트 `num_list`가 주어진다.  
이 안에 **짝수**와 **홀수**의 개수를 각각 구하여  
[짝수 개수, 홀수 개수] 형태의 배열로 return 하라.

- 제한사항  
  - 1 ≤ `num_list`의 길이 ≤ 100  
  - 0 ≤ `num_list`의 원소 ≤ 1,000  

- 반환값: 정수 배열  
  - `[짝수 개수, 홀수 개수]`

## 💡 예시
| num_list | 결과 |
|-----------|--------|
| [1, 2, 3, 4, 5] | [2, 3] |
| [1, 3, 5, 7] | [0, 4] |
