# [Level 0] 특정 문자 제거하기

## 📝 문제 설명 (요약)  
문자열 `my_string`과 문자 `letter`가 주어진다.  
`my_string`에서 `letter`를 **모두 제거한 문자열**을 return 하라.

- 제한사항  
  - 1 ≤ `my_string`의 길이 ≤ 100  
  - `letter`는 `my_string`에 반드시 포함되어 있음  

- 반환값: 문자열  
  - `my_string`에서 `letter`가 빠진 문자열

## 💡 예시
| my_string | letter | 결과 |
|------------|---------|--------|
| "abcdef" | "f" | "abcde" |
| "BCBdbe" | "B" | "Cdbe" |
