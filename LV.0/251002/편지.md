# [Level 0] 편지

## 📝 문제 설명 (요약)
문자열 `message`가 매개변수로 주어질 때,  
편지를 적기 위해 필요한 **가로축 길이(편지지의 최소 가로 길이)**를 구하여 반환하라.  

- 한 글자당 가로 2cm 차지  
- 따라서 결과는 `message.length() * 2`

- 제한사항: 
  - 1 ≤ `message`의 길이 ≤ 50  
- 반환값: 정수 (필요한 가로 길이)
