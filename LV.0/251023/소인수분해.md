# [Level 0] 소인수분해

## 📝 문제 설명 (요약)
정수 `n`이 주어질 때,  
`n`의 **소인수(1을 제외한 소수인 약수)**들을 오름차순으로 담은 배열을 반환하라.  
단, 중복된 소인수는 한 번만 포함한다.

- 제한사항  
  - 2 ≤ `n` ≤ 10,000

- 반환값: 정수 배열  
  - `n`의 소인수들을 오름차순으로 정렬한 배열

## 💡 예시
| n | 결과 |
|---|------|
| 12 | [2, 3] |
| 17 | [17] |
| 420 | [2, 3, 5, 7] |
