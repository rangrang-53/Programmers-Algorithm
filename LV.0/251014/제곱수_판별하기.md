# [Level 0] 제곱수 판별하기

## 📝 문제 설명 (요약)  
정수 `n`이 주어질 때,  
`n`이 **어떤 정수의 제곱**인지 판별하여  
맞으면 `1`, 아니면 `2`를 return 하라.

- 제한사항  
  - 1 ≤ `n` ≤ 1,000,000

- 반환값: 정수  
  - `1` → n이 제곱수일 때  
  - `2` → n이 제곱수가 아닐 때

## 💡 예시
| n | 결과 |
|----|------|
| 144 | 1 |
| 976 | 2 |
