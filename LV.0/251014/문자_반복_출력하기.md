# [Level 0] 문자 반복 출력하기

## 📝 문제 설명 (요약)  
문자열 `my_string`과 정수 `n`이 주어진다.  
`my_string`의 **각 문자를 n번씩 반복한 문자열**을 return 하라.

- 제한사항  
  - 2 ≤ `my_string`의 길이 ≤ 5  
  - 2 ≤ `n` ≤ 10  
  - `my_string`은 알파벳 대소문자로만 구성됨  

- 반환값: 문자열  
  - 각 문자가 n번 반복된 새 문자열

## 💡 예시
| my_string | n | 결과 |
|------------|---|------|
| "hello" | 3 | "hhheeellllllooo" |
| "abc" | 2 | "aabbcc" |
