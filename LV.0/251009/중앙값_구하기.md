# [Level 0] 중앙값 구하기

## 📝 문제 설명 (요약)
정수 배열 `array`가 매개변수로 주어진다.  
`array`의 중앙값을 return 하라.  
(중앙값: 데이터를 오름차순으로 정렬했을 때, 중앙에 위치한 값)

- 제한사항  
  - `array`의 길이는 홀수  
  - 1 ≤ `array`의 길이 ≤ 100  
  - 0 ≤ `array`의 원소 ≤ 1,000  

- 반환값: 정수 (중앙값)
