# [Level 0] 배열 두 배 만들기

## 📝 문제 설명 (요약)
정수 배열 `numbers`가 매개변수로 주어진다.  
`numbers`의 각 원소에 두 배한 값을 담은 **새로운 배열**을 return 하라.

- 제한사항  
  - 1 ≤ `numbers`의 길이 ≤ 1,000  
  - -1,000 ≤ `numbers`의 원소 ≤ 1,000  
- 반환값: 각 원소를 두 배로 만든 정수 배열
