# [Level 0] 최빈값 구하기

## 📝 문제 설명 (요약)
정수 배열 `array`가 매개변수로 주어진다.  
`array` 안에서 **가장 자주 나오는 값(최빈값)**을 return 하라.  
단, 최빈값이 여러 개면 `-1`을 return 한다.

- 제한사항  
  - 0 ≤ `array`의 원소 ≤ 1000  
  - `array`의 길이 ≥ 1  
- 반환값: 정수 (최빈값 or -1)
