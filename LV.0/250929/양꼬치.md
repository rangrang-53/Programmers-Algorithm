# [Level 0] 양꼬치

## 📝 문제 설명 (요약)
양꼬치 1인분은 12,000원, 음료수 1개는 2,000원.
양꼬치 `n`인분과 음료수 `k`개를 먹었다면 총 가격을 계산하여 반환하라.  
단, **양꼬치 10인분을 먹으면 음료수 1개가 서비스**로 제공된다.

- 제한사항: 
  - 0 < n < 1,000  
  - n / 10 ≤ k < 1,000  
- 반환값: 정수 (총 가격)
