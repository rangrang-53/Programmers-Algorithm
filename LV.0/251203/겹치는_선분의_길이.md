# [Level 0] 겹치는 선분의 길이

## 📝 문제 설명 (요약)
정수로 이루어진 **선분 3개**가 주어진다.  
각 선분은 `[start, end]` 형태의 배열로 표현되며,  
이 3개의 선분 중 **겹치는 부분의 총 길이**를 구하는 문제이다.

단, 선분 끝점이 붙어만 있는 경우(예: 끝=2, 다음=2)는  
겹치는 길이에 포함되지 않는다.

---

## 📌 제한사항
- `lines.length = 3`
- 각 `lines[i]`는 길이가 2인 정수 배열
- 선분의 범위는 -100 ≤ x ≤ 100  
- 모든 입력은 정렬되어 있지 않을 수 있음

---

## 💡 반환값
3개의 선분 중 **겹치는 구간들의 총 길이**를  
정수로 반환한다.

---

## 💬 예시

| lines                          | 결과 | 설명 |
|-------------------------------|------|------|
| `[[0,1],[2,5],[3,9]]`         | 2    | 3~5 구간이 2만큼 겹침 |
| `[[ -1,1 ], [ 1,3 ], [ 3,9 ]]`| 0    | 닿기만 하고 실제로 겹치는 구간 없음 |
