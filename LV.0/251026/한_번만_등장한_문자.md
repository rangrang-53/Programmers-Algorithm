# [Level 0] 한 번만 등장한 문자

## 📝 문제 설명 (요약)
문자열 `s`가 주어질 때,  
문자열에서 **한 번만 등장하는 문자들**을  
**사전 순으로 정렬한 문자열**로 반환하라.

- 제한사항  
  - 1 ≤ `s`의 길이 ≤ 1,000  
  - `s`는 소문자로만 구성됨

- 반환값: 문자열  
  - 한 번만 등장한 문자들을 오름차순(사전순)으로 정렬한 결과 문자열  

## 💡 예시
| s | 결과 |
|---|------|
| "abcabcadc" | "d" |
| "abdc" | "abcd" |
| "hello" | "eho" |
