# [Level 0] 배열 자르기

## 📝 문제 설명 (요약)
정수 배열 `numbers`와 정수 `num1`, `num2`가 매개변수로 주어진다.  
`numbers`의 `num1`번째 인덱스부터 `num2`번째 인덱스까지(포함) 자른 배열을 return 하라.

- 제한사항  
  - 2 ≤ `numbers`의 길이 ≤ 30  
  - 0 ≤ `numbers`의 원소 ≤ 1,000  
  - 0 ≤ `num1` < `num2` < `numbers.length`  
- 반환값: 정수 배열 (부분 배열)
