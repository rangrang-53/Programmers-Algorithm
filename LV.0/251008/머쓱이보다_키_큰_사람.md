# [Level 0] 머쓱이보다 키 큰 사람

## 📝 문제 설명 (요약)
머쓱이는 학교에서 키 순으로 줄을 설 때 몇 번째로 서야 하는지 궁금해졌다.  
머쓱이네 반 친구들의 키가 담긴 정수 배열 `array`와 머쓱이의 키 `height`가 매개변수로 주어질 때,  
머쓱이보다 키 큰 사람의 수를 return 하도록 `solution` 함수를 완성하라.

- 제한사항  
  - 1 ≤ `array`의 길이 ≤ 100  
  - 1 ≤ `height` ≤ 200  
  - 1 ≤ `array`의 원소 ≤ 200  
- 반환값: 정수 (머쓱이보다 키 큰 사람의 수)
