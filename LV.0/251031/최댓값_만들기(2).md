# [Level 0] 최댓값 만들기 (2)

## 📝 문제 설명 (요약)
정수 배열 `numbers`가 주어진다.  
이때, 배열 안의 두 수를 곱했을 때 **가장 큰 값**을 반환하라.  
단, 음수도 포함될 수 있다.

- 제한사항  
  - 2 ≤ `numbers`의 길이 ≤ 100  
  - -10,000 ≤ `numbers[i]` ≤ 10,000  

- 반환값  
  - 가장 큰 곱 (int)

## 💡 예시
| numbers | 결과 |
|----------|------|
| [1, 2, -3, 4, -5] | 15 |
| [0, -31, 24, 10, 1, 9] | 240 |
| [10, 20, 30, 5, 5, 20, 5] | 600 |
