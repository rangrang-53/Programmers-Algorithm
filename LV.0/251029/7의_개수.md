# [Level 0] 7의 개수

## 📝 문제 설명 (요약)
정수 배열 `array`가 주어질 때,  
배열에 포함된 모든 숫자를 이어 붙인 문자열에서  
**숫자 7이 등장하는 횟수**를 반환하라.

- 제한사항  
  - 1 ≤ `array`의 길이 ≤ 100  
  - 0 ≤ `array[i]` ≤ 100,000  

- 반환값:  
  - 배열의 모든 숫자를 문자열로 합친 뒤,  
    등장하는 `'7'`의 총 개수

## 💡 예시
| array | 결과 |
|--------|------|
| [7, 77, 17] | 4 |
| [10, 29] | 0 |
