# [Level 0] 중복된 숫자 개수

## 📝 문제 설명 (요약)
정수 배열 `array`와 정수 `n`이 주어진다.  
`array` 안에 `n`이 몇 개 포함되어 있는지 개수를 반환하라.

- 제한사항  
  - 1 ≤ `array`의 길이 ≤ 100  
  - 0 ≤ `array`의 원소 ≤ 1,000  
  - 0 ≤ `n` ≤ 1,000  

- 반환값  
  - 정수 (배열 안에 `n`이 등장한 횟수)

## 💡 예시
| array | n | 결과 |
|--------|---|------|
| [1, 1, 2, 3, 4, 5] | 1 | 2 |
| [0, 2, 3, 4] | 1 | 0 |
