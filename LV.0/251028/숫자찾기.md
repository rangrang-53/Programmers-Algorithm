# [Level 0] 숫자 찾기

## 📝 문제 설명 (요약)
정수 `num`과 `k`가 주어진다.  
`num`을 이루는 숫자 중에서 `k`가 **처음 등장하는 위치(인덱스)**를 반환하라.  
단, **1부터 시작하는 인덱스**로 계산한다.  
만약 `k`가 존재하지 않으면 `-1`을 반환하라.

- 제한사항  
  - 0 < `num` < 1,000,000  
  - 0 ≤ `k` < 10

- 반환값: 정수  
  - `k`가 처음 등장한 위치 (1-based index)  
  - 존재하지 않으면 `-1`

## 💡 예시
| num | k | 결과 |
|------|---|------|
| 29183 | 1 | 3 |
| 232443 | 4 | 4 |
| 123456 | 7 | -1 |
