# [Level 0] 약수 구하기

## 📝 문제 설명 (요약)
정수 `n`이 주어질 때,  
`n`의 **모든 약수**를 **오름차순 배열**로 반환하라.

- 제한사항  
  - 1 ≤ `n` ≤ 10,000

- 반환값: 정수 배열  
  - `n`의 약수들을 오름차순으로 담은 배열

## 💡 예시
| n | 결과 |
|---|------|
| 24 | [1, 2, 3, 4, 6, 8, 12, 24] |
| 29 | [1, 29] |
