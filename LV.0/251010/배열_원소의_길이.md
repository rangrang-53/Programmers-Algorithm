# [Level 0] 배열 원소의 길이

## 📝 문제 설명 (요약)  
문자열 배열 `strlist`가 매개변수로 주어진다.  
각 문자열의 **길이**를 원소로 가지는 정수 배열을 return 하라.

- 제한사항  
  - 1 ≤ `strlist`의 길이 ≤ 100  
  - 1 ≤ 각 문자열의 길이 ≤ 100  

- 반환값: 정수 배열  
  - 각 문자열의 길이를 순서대로 담은 배열

## 💡 예시
| strlist | 결과 |
|----------|------|
| ["We", "are", "the", "world!"] | [2, 3, 3, 6] |
| ["I", "Love", "Programmers."] | [1, 4, 12] |
