# [Level 0] 배열의 유사도

## 📝 문제 설명 (요약)  
두 문자열 배열 `s1`, `s2`가 주어진다.  
두 배열에 **공통적으로 들어있는 문자열의 개수**를 return 하라.

- 제한사항  
  - 1 ≤ `s1`, `s2`의 길이 ≤ 100  
  - 1 ≤ 각 문자열의 길이 ≤ 10  
  - `s1`, `s2`는 알파벳 소문자로만 구성됨  
  - 중복된 원소는 없음

- 반환값: 정수  
  - 두 배열의 **유사도(공통 원소의 개수)**
