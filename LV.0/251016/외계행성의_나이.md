# [Level 0] 외계행성의 나이

## 📝 문제 설명 (요약)  
머쓱이는 외계행성에 살고 있다.  
이 행성에서는 나이를 **알파벳 소문자**로 표현한다.  

나이를 나타내는 규칙은 다음과 같다:  
- 0 → a  
- 1 → b  
- 2 → c  
- ...  
- 9 → j  

정수 `age`가 주어질 때,  
이 행성식으로 변환한 **문자열**을 return 하라.

- 제한사항  
  - 0 ≤ `age` ≤ 1,000  
  - 나이는 정수  

- 반환값: 문자열  
  - `age`의 각 자리를 알파벳으로 바꾼 결과

## 💡 예시
| age | 결과 |
|------|------|
| 23 | "cd" |
| 51 | "fb" |
| 100 | "baa" |
