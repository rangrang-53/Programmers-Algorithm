# [Level 0] 진료 순서 정하기

## 📝 문제 설명 (요약)  
정수 배열 `emergency`가 주어진다.  
각 원소는 환자의 **응급도**를 나타낸다.  

응급도가 **높을수록 우선순위가 높다**.  
즉, 값이 클수록 더 높은 순위를 받아야 한다.  

각 환자의 우선순위를 담은 배열을 return 하라.  
(같은 응급도는 주어지지 않는다.)

- 제한사항  
  - 3 ≤ `emergency`의 길이 ≤ 10  
  - 1 ≤ `emergency`의 원소 ≤ 100  

- 반환값: 정수 배열  
  - 각 환자의 순위를 원래 순서대로 담은 배열

## 💡 예시
| emergency | 결과 |
|------------|--------|
| [3, 76, 24] | [3, 1, 2] |
| [1, 2, 3, 4, 5, 6, 7] | [7, 6, 5, 4, 3, 2, 1] |
| [30, 10, 23, 6, 100] | [2, 4, 3, 5, 1] |
