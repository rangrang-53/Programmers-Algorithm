# [Level 0] 가까운 수

## 📝 문제 설명 (요약)
정수 배열 `array`와 정수 `n`이 주어진다.  
`array`에서 `n`과 **가장 가까운 수**를 찾아 반환하라.  
가장 가까운 수가 둘 이상이면 **더 작은 수**를 반환한다.

- 제한사항
  - 1 ≤ `array.length` ≤ 100
  - -100 ≤ `array[i]` ≤ 100
  - -100 ≤ `n` ≤ 100

- 반환값: 정수  
  - `n`과 가장 가까운(동일 거리면 더 작은) 값

## 💡 예시
| array | n | 결과 |
|------|---|-----|
| [3, 10, 28] | 20 | 28 |
| [10, 11, 12] | 13 | 12 |
| [4, 1, 2] | 3 | 2 |
