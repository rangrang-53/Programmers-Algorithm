# [Level 0] 모스부호 (1)

## 📝 문제 설명 (요약)  
머쓱이는 친구들에게 메시지를 모스부호로 보낸다.  
모스부호 규칙표가 주어지고,  
모스부호로 표현된 문자열 `letter`가 입력된다.  

이때 `letter`를 해독해 **영문 소문자 문자열**로 바꿔서 return 하라.

- 제한사항  
  - 1 ≤ `letter`의 길이 ≤ 1,000  
  - 공백으로 모스부호가 구분되어 있음  
  - 모스부호 표는 다음과 같음 👇

| 알파벳 | 모스부호 |
|---------|-----------|
| a | .- |
| b | -... |
| c | -.-. |
| d | -.. |
| e | . |
| f | ..-. |
| g | --. |
| h | .... |
| i | .. |
| j | .--- |
| k | -.- |
| l | .-.. |
| m | -- |
| n | -. |
| o | --- |
| p | .--. |
| q | --.- |
| r | .-. |
| s | ... |
| t | - |
| u | ..- |
| v | ...- |
| w | .-- |
| x | -..- |
| y | -.-- |
| z | --.. |

- 반환값: 문자열  
  - 모스부호를 해독한 알파벳 문자열

## 💡 예시
| 입력 | 출력 |
|------|------|
| `"... --- ..."` | `"sos"` |
| `".--. -.-- - .... --- -."` | `"python"` |
